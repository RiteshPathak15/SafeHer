# Rakshika-Ai

Rakshika-Ai is a web-based women safety analytics system that analyzes crime data related to women in India. The system collects district-level crime data and analyzes patterns such as crime trends, high-risk districts, and crime categories.

## 1. Introduction
Women’s safety is a major concern in many parts of India. Crime data related to women is available through government sources, but it is often difficult for the public to interpret and analyze this information effectively. There is a need for a system that can analyze crime statistics and present them in a clear and meaningful way.

Rakshika-Ai is designed to analyze district-level crime data related to women across India. The system processes government crime datasets to identify patterns such as crime trends, high-risk districts, and crime categories. Machine learning techniques are used to predict risk levels in different districts.

The results are presented through interactive dashboards, charts, and maps, enabling users to easily understand crime patterns. Additionally, the system includes a global chat and an emergency feature that allows users to share their location if they feel unsafe.

## 2. Objectives
- Analyze crime data related to women across different districts in India.
- Identify crime trends and high-risk areas using data analysis.
- Predict district-level risk levels using machine learning models.
- Present crime insights through interactive dashboards and visualizations.
- Provide a global chat interface for real-time user conversations.
- Allow users to share their location during unsafe situations for assistance.

## 3. Scope
The Rakshika-Ai system has several applications and benefits:
- Helps users understand crime trends and identify unsafe areas.
- Supports awareness regarding women’s safety in different regions.
- Provides data-driven insights for better decision-making.
- Assists authorities and researchers in analyzing crime patterns.
- Enables emergency location sharing during unsafe situations.

Future enhancements:
- Integrating real-time crime data.
- Mobile application support.
- Advanced AI-based prediction models.
